# Desafio – Processo Seletivo LabSEC 2025.2

Este repositório reúne as duas páginas desenvolvidas como parte do desafio de front-end para o processo seletivo LabSEC 2025.2.

## Links do projeto
- **Repaginação do site do LabSEC (Design System gov.br):**  
  https://desafio-front-end-theta.vercel.app/

- **Página original sobre segurança de senhas:**  
  https://desafio-front-end-gpxc.vercel.app/
---

## 1. Introdução

O desafio consistiu na implementação de duas páginas web independentes.

A primeira proposta envolveu a criação de uma nova versão do site oficial do LabSEC, seguindo rigorosamente o Design System do gov.br, que define padrões de identidade visual, acessibilidade e usabilidade para interfaces digitais do governo federal. O objetivo foi reproduzir essa experiência de forma fiel e funcional.

A segunda parte do desafio pediu o desenvolvimento de uma página original relacionada à área de Segurança em Computação. Para essa etapa, foi criada uma página interativa sobre segurança de senhas, com foco em criatividade, conteúdo e liberdade de design.

---

## 2. Tecnologias Utilizadas

### Repaginação do site do LabSEC
- Vite  
- React  
- Typescript (TSX)  
- TailwindCSS  
- Rawline (tipografia oficial)  
- Font Awesome (ícones)  
- UIKit Web e componentes do Design System gov.br  
- Figma para referência visual

O processo de desenvolvimento incluiu:
- Estudo do design system gov.br  
- Análise dos componentes disponíveis  
- Criação do layout no Figma  
- Implementação dos componentes e da página em TSX  

---

### Página original sobre segurança de senhas
- React  
- JSX (para facilitar manipulação direta do Canvas)  
- Canvas API para animação interativa  
- Bibliotecas e componentes modernos (animações, slider, ícones)

Essa página foi desenvolvida com liberdade criativa, com o objetivo de demonstrar domínio de front-end, senso estético, interatividade e criação de experiências modernas.
